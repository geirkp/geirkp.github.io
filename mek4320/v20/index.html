Assignment 2; preliminary description

In the assignment we utilize a simple Boussinesq model. A description with examples are given in the page linked below. While it is possible to compile the code it is recommended to copy the "sbouss" executable which should run on all 64 bit Linux implementations.
  • The Boussinesq model.
  • A short treatise on the model.
  • Problems related to the model.
You are required to do problems 1ab, 2, 4 and 5 (may be changed). Problems 1ab introduce you for the concept of numerical dispersion and will not be a part of the submission/presentation of the assignment. The rest is, essentially, a series of simulations for which you must assess the results and relate them to each other and to subjects from the syllabus. If you organize your work in a structured manner, making the simulations should not be too time consuming.
Normally the students will collaborate on doing the assignment. However, this may be difficult now. Besides, it is important that all students make themselves familiar with all the problems, as some of these will be the basis for questions at the final exam.

Deadline and details are still to be decided. A few days prior to the deadline each student will be given a part of the assignment, for which they will make a presentation which is sent to the lecturer and that will be dicussed on, for instance, Discord or Skype. Beyond this, there is no submission of any written report or program snippets.
