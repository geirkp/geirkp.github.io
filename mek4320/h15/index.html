Assignment 2

In the assignment we utilize a simple Boussinesq model. A description with examples are given in the page linked below. While it is possible to compile the code it is recommended to copy the "sbouss" executable which should run on all 64 bit Linux implementations.
  • The Boussinesq model.
  • A short treatise on the model.
  • Problems related to the model.
You are required to problems 2-6 . This is, essentially, a series of simulations for which you must assess the results and relate them to each other and to subjects from the syllabus. If you organize your work in a structured manner, making the simulations should not be too time consuming. The deadline is Thursday, November 12th, 16.00.
