Compulsory assignments.

Assignment 1

Two problems on properties of the linear wave mode.
This assignment makes you probe further into the shallow/deep water issues and the validity of linearization for gravity waves.
Deadline and way of submission are still to be decided.

Assignment 2

In the assignment we utilize a simple Boussinesq model. A description with examples are given in the page linked below. While it is possible to compile the code it is recommended to copy the "sbouss" executable which should run on all 64 bit Linux implementations.
  • The Boussinesq model.
  • A short treatise on the model.
  • Problems related to the model.
You are required to do problems 2-6 . This is, essentially, a series of simulations for which you must assess the results and relate them to each other and to subjects from the syllabus. If you organize your work in a structured manner, making the simulations should not be too time consuming.
The students will work in groups. Each group will do all parts of the assignment. It is also important that all students within each group make themselves familiar with all the problems, as some of these will be the basis for questions at the final exam.
The day for the presentation is 6/5. A few days prior to this deadline each group will be given a part of the assignment, which they will present in class at this day. The presentation, and ensuing discussion, will be assessed. There is no submission of any written report or program snippets for the students that do presentations.
