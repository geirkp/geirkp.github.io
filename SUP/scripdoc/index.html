List of scripts and programs

Program Directory Purpose
apbonew.sh /hom/geirkp/bouss/trappe/bin Reproduces a boslide simulation
bounl.sh /hom/geirkp/lagbouss/bin Computes boundary layers from a knaai simulation.
cleansepar.pe /hom/geirkp/bin Use (example): cleansepar.pe gg.par > res.par. Reads parameterfile gg.par and write it to res.par omitting multiple settings of parameters. The last setting is the one kept.
Com.pe /hom/geirkp/bouss/trappe/bin Produces COMMIT propagation files from propagation files created in gglo simulation
comtogp /hom/geirkp/lib/i386 Translates COMMIT propagation files to gphov format
cusnit.pe /hom/geirkp/bin Extract general transects from gphov files.
derv /hom/geirkp/bin Computes derivatives by polynomial regression.
dfi2txt, dfi2txt32 /hom/geirkp/lab/bin or /mn/anatu/hydrolab-data/josteiko/digiflow Converts binary digiflow files to ascii.
diffil.pe /hom/geirkp/bin Makes linear combinations of data rom two files by means of spline interpolation
ddrun.pe /hom/geirkp/lagbouss/bin Makes oblique run-up simulations by ddsta* programs
Dsor.pe /hom/geirkp/bouss/trappe/test Makes test source for gglo program
etpcN /hom/geirkp/bouss/lib/i386 Fortran program that extracts a gphov depth matrix from a downloaded ETOPO1 dataset in netCDF.
felt.sh /hom/geirkp/laplace/rusource/bin Produces velocity fields from the output of the boundary integral program.
Fprod /hom/geirkp/solitons/i386 Fortran program that produces solitary wave profiles from Fenton's fourth order solution.
fullsol /hom/geirkp/bin Shell script that produces solitary wave profiles from the Tanaka solution, by solp, or Fenton's solution, by Fprod.
gbimpar.pe /hom/geirkp/laplace/rusource/bin Synchronization of Lagrangian and BIM runup models with data from experiments or other models.
genkurv /hom/geirkp/bin/ Swift plotting of 2D curves by plotxy or gnuplot. Can be combined with pakk.pe to make latex figures with legends and labels in latex code. List of symbols in plotxy mode.
genpil.pe /hom/geirkp/bin/ Invokes pilplott.pe, genkurv,peplot.pe and pakk.pe to produce vector plots
getnew.pe /hom/geirkp/gitec/bin/ Extracts parameters from a parameterfile
gglo* /hom/geirkp/bouss/trappe/i386 Fortran program for global Boussinesq solver.
gglparse.pe /hom/geirkp/bouss/trappe/bin Script for running global Boussinesq solver.
ggrestart.pe /hom/geirkp/bouss/trappe/bin Prolongation of Boussinesq solver run through gglparse.pe .
gkontur.pe /hom/geirkp/bin/ Uses Gsharp to produce countour and 3D plots
gparvar.pe /hom/geirkp/bin Extracts lists ov values from parameter files inn distributed folders.
gpdiff.pe /hom/geirkp/bin Linear combinations on single-dataset gphov files.
gpmask.pe /hom/geirkp/bin Transfer a mask from one single-dataset gphov file to another.
gpregrid.pe /hom/geirkp/bin Employ refgp to regrid a field.
gpslag.pe /hom/geirkp/bin Probes the format of a gphov file.
gp2xyval.pe /hom/geirkp/bin Converts a gphov formatted file to a column format x y 'fieldvalue'
gptocom /hom/geirkp/lib/i386 Translates gphov files to COMMIT propagation files
gsersplit.pe /hom/geirkp/lagbouss/bin Split time series from knaai (Lagrangian Boussinesq model)
gskred.pe /hom/geirkp/bouss/slide/bin Runs series of simulations of slide generated waves(boslide model)
gsplit /hom/geirkp/bin Extracts columns in genkurv style, but without plotting
initialperf.sh /hom/geirkp/SUP/experiments Initiation of paddle iteration
itperf.sh /hom/geirkp/SUP/experiments Paddle iteration
Kjfilt /hom/geirkp/lib/i386 Fortran program for surface respons to earthquake sources.
knaai* /hom/geirkp/lagbouss/i386 Lagrangian Boussinesq model with runup on sloping beaches
lru* /hom/geirkp/laplace/rusource/i386 Boundary integral model
lagsersplit.pe /hom/geirkp/lagbouss/bin Split Lagrangian time series from knaai (Lagrangian Boussinesq model)
lagtser.pe /hom/geirkp/SUP/experiments Produces paddle motion voltages for labview from 'tx' or 'tV' input.
laptobous.pe /hom/geirkp/lagbouss/bin Makes Boussinesq simulation from BIM data.
linset.pe /hom/geirkp/bin Converting irregular, but monotonic, xy files to regular xy files or gphov files.
mulprf /hom/geirkp/boelge/bounlay/i386 Boundary layer model
minterpol.pe /hom/geirkp/bin Linear interpolation in data-file
msquare.pe /hom/geirkp/bin Linear regression applied to a file with irregular data.
newrun /hom/geirkp/bin A script to run snippets of bash code associated with a project.
nDut /hom/geirkp/SUP/experiments/i386 Fortran program for acceleration calculations. Compiled only for linux.
onetotwo /hom/geirkp/lib/i386 Extend a plane profile to a toD gphov file
pakk.pe /hom/geirkp/gitec/bin Arrange one ore more ps figures, or latex text files, into a picture environment.
peplot.pe /hom/geirkp/bin/graf Enkelt plotteverktøy.
pilplott.pe /hom/geirkp/bin/graf Creates inputfiles for peplot.pe that produce vector-field plots
pdatpros.pe /hom/geirkp/SUP/experiments Converts a xyuv file to gphov format.
pkat /hom/geirkp/bin Concatenates and manipulates on gphov files. Invokes akop from main program /hom/geirkp/lib/kopsource/genkop.f.
poly /hom/geirkp/lib/i386 Fortran program that computes smooth data from et polygon by means of splines.
reorfig.pe /hom/geirkp/bin Extracts the picture environment from a file generated by pakk.pe. Produce a single ps file from everything in this environment and wraps the new ps with original caption into a new tex file by pakk.pe
revert.pe /hom/geirkp/bin Shift the sequence of lines in a file . Use: revert.pe oldfile > newfile
roterser.pe /hom/geirkp/bin Rotates and transformes column files.
rotxy.pe /hom/geirkp/lagbouss/bin Rotates and transformes xy files from Boussinesq or BIM runup models to coordinates pararallel and normal to the beach plane. Origin for the rotated system in the equilibrium shoreline position.
rurun.pe /hom/geirkp/lagbouss/bin Sets up simulations with knaai*.
slidmat.pe /hom/geirkp/bouss/slide/bin Generation of matrices for seabed and slide-bodies from the output of boslide.new.
sniffcdf /hom/geirkp/lib/i386 Browses netCDF files.
solcomp /hom/geirkp/bin Shell script that compares the data on a specified file with solitary wave profiles. Invokes soliprod
solfelt.sh /hom/geirkp/laplace/rusource/bin Shell script that produces velocity and pressure fields for a solitary wave from full potential theory.
soliprod /hom/geirkp/bin Generates solitary wave data.
solp /hom/geirkp/solitons/i386 Fortran program that generates solitary wave data from Tanaka's solution.
splitl.pe /hom/geirkp/bouss/trappe/bin Extracts rows and columns from gphov files.
splitser.pe /hom/geirkp/bouss/trappe/bin Processes time series from gglo.
textab.pe /hom/geirkp/bin Converts a txt table into latex.
timsenew.sh /hom/geirkp/SUP/experiments Produces unix 2-column files from labview output. Applies smoothing and regression to make time series suitable for calculation of paddle velocities for input in models.
tserslice.pe /hom/geirkp/lagbouss/bin Interpolation of time-series for given times.
ufromg* /hom/geirkp/labproj/progs/i386 Fortran program for generaton of velocity series from surface elevations.
uls /hom/geirkp/tsunamisource/okada/i386 Fortran program for Okadas formula.
unslopgp.pe /hom/geirkp/bin Produces gphov ascii file from sloppy file with wrong number of blanks in records.
Uturn.pe /hom/geirkp/bin Swaps a gphov file for velocity.
