Extension to 2D

onetwoto is a compiled f77 code.

The code

directory: /hom/geirkp/lib
files: one2two.f
additional libraries: biblGF8.a, diaGF8.a in /hom/geirkp/lib/*
makefile: i386/monetoto
executable: i386/onetotwo

Use

An input file is read on the form 
!give 1D data file
aa.dat
!give position of s=0
-5 0
!give angle between s and x axis in deg
31
!give ll position of field
-7 2
!give increments
0.8 1.3
!give grid size
49 50
and a file ufelt is produced.

Note

May not be all that well tested.
