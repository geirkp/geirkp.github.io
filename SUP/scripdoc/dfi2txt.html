A digiflow to ascii converter.

This page is a documentation on a .dfi to text filter for binary data-files from DigiFlow. Author of script: Jostein Kolaas.

Use

Example (32 bits code) 
~/lab/bin/dfi2txt32 piv003415.dfi
Converting piv003415.dfi -> piv003415.txt
The heading of the generated file looks like 
64 512 4
 -0.0000000000000000e+00 -0.0000000000000000e+00  8.0000000000000000e+00  1.0000000000000000e+00
 -0.0000000000000000e+00 -0.0000000000000000e+00  2.4000000000000000e+01  1.0000000000000000e+00
 -0.0000000000000000e+00 -0.0000000000000000e+00  4.0000000000000000e+01  1.00000
The first two numbers on the first line are array sizes (nx and ny). The third is number of fields. In this case 4 means u v x and y.

kommenter (#) linjene under "# matlab 64 bit linux" og
fjern kommenteringen under "# matlab 32 bit linux". og så kjøre:
$ Make clean
$ Make
