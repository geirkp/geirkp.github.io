Program for smoothing of polygon data

The program reads the points of a polygon in x and y together with a transition interval in x. Data are then generated where the polygon is smoothed by a polynomial, giving continuous second derivatives, in the transition intervals. The program will not work properly if the transition intervals do overlap.

Use

An input file to the program may read (see example) 
!give file
korn.dat
!give interval
1 11
!give n
100
!give undef value/-666/

!give tolerance at boundaries/1.0e-7/
Here the file korn.dat contains three columns, x , y and transition interval. An interval, not necessarily coinciding with the interval in the x-data -- even though this is the default , is given. The number of grid points in the smooth data is set to 100, while x values outside the polygon data are assigned y-values as given by undef (default -666). All outside points within tolerance is redefined as interior points.
